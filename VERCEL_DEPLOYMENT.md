# Deploy to Vercel CLI

```bash
npm install -g vercel
vercel
vercel --prod
```

That's it! 

- `npm install -g vercel` - Install Vercel CLI
- `vercel` - Deploy preview
- `vercel --prod` - Deploy to production

Add your Supabase env vars when prompted:
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
